This project is created with Django(Python).
It is still under development.
It is designed in accordance with Google Chrome.
It requires an active Internet Connection.

To run it follow these steps:-
1. In the root directory of the project(where manage.py is located), open cmd by 
   typing cmd in the address bar.
2. Type the following commands in the cmd:-
	1. cd python
	2. cd scripts
	3. activate
	4. cd..
	5. cd..
	6. python manage.py runserver
4. Go to the given(displayed in cmd) url
5. Enjoy

Default superuser details are:-
username = admin
password = admin

1. django
